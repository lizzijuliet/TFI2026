## Programa Utilizado
PSeInt y Google Colab con Python

## Descripción
Este repositorio contiene el Trabajo Final Integrador de la materia Programación.

El trabajo está compuesto por diez ejercicios. Los ejercicios fueron desarrollados en PSeInt, aplicando estructuras secuenciales, condicionales, repetitivas, vectores, matrices, funciones y subprocesos.

Además, el ejercicio 10, Redes y Comunicación, también fue desarrollado en Python utilizando Google Colab. En este ejercicio se representa una red direccionada de ocho nodos de telecomunicaciones, donde cada nodo posee comunicación directa con otros dos nodos. También se muestra la matriz de adyacencia y una representación gráfica de la red.
